# 🏭 Gidrobas — Дипломный проект

## 📌 Описание проекта

Данный проект представляет собой веб-сайт компании **Gidrobas**, занимающейся производством, продажей гидравлических комплектующих для спецтехники, и ремонта гидроцилиндров.

Сайт разработан в рамках дипломной работы и предназначен для:

* демонстрации продукции
* ознакомления клиентов с компанией
* предоставления контактной информации

---

## 🚀 Используемые технологии

* **React (Vite)** — фронтенд
* **JavaScript (ES6)** — логика приложения
* **CSS3** — стилизация интерфейса
* **HTML5** — структура страниц

---

## 🧩 Структура проекта

```
- App - корневой компонент приложения
- Header - шапка сайта с навигацией
- Footer - подвал сайта
- Hero - главный баннер на стартовой странице
- Catalog - компонент отображения каталога товаров
- CatalogPage - страница каталога с полным функционалом
- ProductPage - страница отдельного товара
- Cart - корзина, оформление заказа, история заказов
- AuthModal - модальное окно авторизации и регистрации
- Contacts - страница с контактной информацией
- RepairService - страница или блок с информацией о ремонте

---

## 📦 Функционал сайта

### 🔹 Каталог товаров

* Отображение гидравлических комплектующих

### 🔹 Контакты
